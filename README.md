HOW TO INSTALL AND USE : 

Step 1: Clone the repository
git clone https://github.com/harshgupptaa/simple-chrome-activity-logger.git

Step 2: Open Chrome Extensions page
Open Chrome and go to: chrome://extensions

Step 3: Enable Developer Mode
Turn ON Developer mode from the top-right corner.

Step 4: Load the extension
Click Load unpacked and select the folder:
simple-chrome-activity-logger

Step 5: Open the extension
Click the extension icon from the Chrome toolbar.

Step 6: Inspect logs
Go to chrome://extensions

a.Click Details on the extension

b.Click Inspect views then popup.html

c.Open Console

Step 7: Perform actions
Type in the input field and click the button.
All activities, events, and content changes will appear in the Console logs.
